# Azure SQL Setup and Usage Tutorial

This tutorial will guide you through the process of setting up and using Azure SQL.

## Prerequisites

Before you start, make sure you have installed:

- SQL Server Management Studio (latest version)
- BCP and SQLCMD (latest version)

## Step 1: Sign in to the Azure portal

Sign in to the Azure portal at https://portal.azure.com/.

## Step 2: Create a new SQL Database

1. On the Azure portal menu or from the Home page, select **Create a resource**.
2. On the **Create a resource** page, search for **SQL Database** and select it.
3. In the **SQL Database** pane, select **Create**.

## Step 3: Fill out the Basics tab

Fill out the mandatory information required on the Basics tab. This is a minimum set of information required to provision a SQL Database.

## Step 4: Query the Database

After the database is created, you can query the database using SQL Server Management Studio or Azure Data Studio.

## Step 5: Clean up resources

When you're done with the database, you can delete it to avoid incurring any further costs.

For more detailed instructions, you can refer to the official Azure SQL Database quickstart guide [here](https://learn.microsoft.com/en-us/azure/azure-sql/database/single-database-create-quickstart?view=azuresql).

Please note that the actual steps may vary slightly depending on the current Azure portal interface and the specific requirements of your project.
